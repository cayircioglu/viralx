# ViralX

Turkiye'nin Ilk AI Destekli Viral Tweet Uretici

## Ozellikler
- Tweet ve Thread uretimi
- 8 Master Persona
- 45+ Hook sablonu
- Virallik analizi
- Reply yazari
- Screenshot export
- CSV toplu uretim
- Light/Dark tema
- Riskli icerik filtresi

## Kurulum
1. `pip install -r requirements.txt`
2. `.env` dosyasina `GEMINI_API_KEY=your_key` ekle
3. `python app.py` ile calistir
